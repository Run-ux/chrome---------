关于API密钥

本插件支持多种翻译和解释API服务，您需要获取相应的API密钥：

  • OpenAI API - 提供高质量的翻译和解释服务
  • Google Cloud Translation API - 专业的翻译服务

请注意保护您的API密钥安全，不要与他人分享。
