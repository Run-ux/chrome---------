智能划词翻译助手 - 使用帮助

插件介绍

智能划词翻译助手是一款功能强大的Chrome浏览器插件，它可以帮助您快速翻译和解释网页上的文本内容。无论您是在阅读外语文章、学习新语言，还是需要理解专业术语，本插件都能为您提供便捷的帮助。

主要功能
  • 划词翻译：选中文本后一键翻译
  • 内容解释：获取选中文本的详细解释和分析
  • 右键菜单：通过右键菜单快速访问翻译和解释功能
  • 多语言支持：支持多种目标语言的翻译
  • 自定义设置：根据个人需求自定义插件行为

使用方法

基本使用

  1. 在网页上选中您想要翻译或解释的文本
  2. 选中后，会在文本附近出现一个蓝色的翻译按钮
  3. 点击翻译按钮，将显示翻译结果弹窗
  4. 在弹窗中，您可以选择「翻译」或「解释」功能

使用右键菜单

  1. 在网页上选中文本
  2. 右键点击选中的文本
  3. 在右键菜单中选择「翻译选中文本」或「解释选中文本」
  4. 结果将在弹窗中显示
提示

您可以通过点击插件图标打开弹出窗口，快速访问插件功能和设置。

设置选项

您可以通过以下方式访问设置页面：

  • 点击插件图标，然后点击「设置」按钮
  • 在Chrome扩展管理页面中找到本插件，点击「选项」

可用设置

  • API密钥：设置用于翻译和解释服务的API密钥
  • 目标语言：选择翻译的目标语言
  • 自动翻译：启用后，选中文本将自动显示翻译结果
  • 显示翻译按钮：控制是否在选中文本旁显示翻译按钮

常见问题

为什么需要API密钥？

本插件使用第三方API服务提供翻译和解释功能，需要API密钥进行身份验证。您可以从OpenAI或Google Cloud等服务提供商获取API密钥。

如何获取API密钥？

您可以通过以下方式获取API密钥：

  • OpenAI API：访问 OpenAI平台 注册账号并创建API密钥
  • Google Cloud Translation API：访问 Google Cloud 创建项目并启用Translation API
插件不显示翻译按钮怎么办？

请检查以下几点：

  • 确保在设置中启用了「显示翻译按钮」选项
  • 尝试重新加载网页
  • 检查是否有其他扩展与本插件冲突
  • 确保您已经选中了文本（按钮只在选中文本后显示）
翻译或解释结果不准确怎么办？

翻译和解释的准确性取决于所使用的API服务。如果结果不准确，您可以：

  • 尝试使用不同的API服务
  • 确保选中的文本清晰完整
  • 对于专业术语或特定领域的内容，可能需要专业的翻译服务
插件支持哪些语言？

本插件支持多种语言的翻译，包括但不限于：简体中文、英语、日语、韩语、法语、德语、西班牙语和俄语。具体支持的语言取决于您使用的API服务。
